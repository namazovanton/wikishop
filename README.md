# «Викишоп»

---
## Описание проекта
Интернет-магазин запустил новый сервис - теперь пользователи могут редактировать и дополнять описания товаров, предлагать свои правки и комментировать изменения других. Необходимо разработать инструмент, который будет искать токсичные комментарии и отправлять их на модерацию. 

---
## Цель исследования
Построить модель классификации комментариев на позитивные и негативные


---
## Библиотеки
```javascript
- Pandas
- Catboost
- LogisticRegression
- AutoTokenizer, AutoModel, AutoConfig
- Numpy
- Matplotlib
- Sklearn
- GridSearchCV
- Statsmodels
```

## Набор данных
В распоряжении набор данных с разметкой о токсичности правок.
```javascript
text - содержит текст комментария
toxic — целевой признак.
```
